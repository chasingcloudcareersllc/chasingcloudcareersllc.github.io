# CLAUDE.md

## Quick Reference

- **Stack**: Next.js 16, React 19, Tailwind CSS 4, TypeScript
- **Build**: `npm run build` (static export to `out/`)
- **Dev**: `npm run dev` (fetches blog + learn content first)
- **Lint**: `npm run lint`
- **Type check**: `npm run type-check`
- **Blog content**: Fetched from `chasingcloudcareersllc/chasingcloudcareersllc-blog` at build time
- **Learn content**: Fetched from `chasingcloudcareersllc/chasingcloudcareersllc-learn` at build time
