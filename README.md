# Chasing Cloud Careers

> **Free tech education, mentorship, and training for aspiring cloud professionals**

Welcome to **Chasing Cloud Careers** - providing public speaking on technical education, mentorship, and tech bootcamp cohorts with free self-service training materials.

## What is Chasing Cloud Careers?

Chasing Cloud Careers LLC provides free, accessible training materials and learning paths for anyone looking to build a career in cloud computing, AI, or DevOps. We believe quality tech education should be available to everyone.

### Our Focus

- **Free Training**: All training materials and learning paths are completely free and self-service
- **Hands-On Learning**: Practical, project-based learning for real-world tech roles
- **Mentorship**: Public speaking on technical education and career development
- **Community**: Tech bootcamp cohorts for collaborative learning

## What We Offer

- **[Learning Paths](https://chasingcloudcareersllc.github.io/learn/)** - Free self-service training materials for cloud, AI, and DevOps
- **[Blog](https://chasingcloudcareersllc.github.io/blog/)** - Insights and tips on tech careers and continuous learning

## Technology

- **Framework**: [Next.js 16](https://nextjs.org/) with static export
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Language**: TypeScript
- **Hosting**: GitHub Pages at [chasingcloudcareersllc.github.io](https://chasingcloudcareersllc.github.io)
- **Blog**: Separate [chasingcloudcareersllc-blog](https://github.com/chasingcloudcareersllc/chasingcloudcareersllc-blog) repo, fetched at build time
- **Learn**: Separate [chasingcloudcareersllc-learn](https://github.com/chasingcloudcareersllc/chasingcloudcareersllc-learn) repo, fetched at build time

## Development

```bash
# Install dependencies
npm install

# Start development server (fetches blog + learn content first)
npm run dev

# Build for production
npm run build

# Type check
npm run type-check

# Lint
npm run lint
```
